# Sam's Wedding

A combined wedding demo with two main applications:

- **Wedding Planning with SAM** — a multi-agent wedding planning system built on Solace Agent Mesh.
- **Wedding Photo Sharing** — an event photo distribution platform using Solace PubSub+ for workflow orchestration and email delivery.
